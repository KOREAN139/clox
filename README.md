# clox
Implemetation of [lox interpreter](http://www.craftinginterpreters.com/contents.html) in C

## DEVELOPMENT STATE
Implemented ~4.5 by far.

## HOW TO USE
###Install
`git clone https://github.com/KOREAN139/clox`
###Build
`make`
###Execute
`./clox [script]`
###Clean
`make clean`
