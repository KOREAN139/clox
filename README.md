# clox
Implemetation of [lox interpreter](http://www.craftinginterpreters.com/contents.html) in C

## CURRENT DEV STATE
Implement Chapter 14, challenge 2, 3 (hardcore mode) by far.

## HOW TO USE

### Install
`git clone https://github.com/KOREAN139/clox`

### Build
`make`

### Execute
`./clox [script]`

### Clean
`make clean`
